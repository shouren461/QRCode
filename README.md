项目介绍:

1,创建功能:采用策略工厂模式(Strategy Factory)来解耦不同二维码类型的协议编码和表单配置，在输入界面通过FormAdapter来实现动态渲染列表视图

2，历史记录:数据库采用Room数据结构，创建HistoryRecord实体类，通过将滚动视图类型分为标题项(Header)和记录项(Record),来实现记录项的收藏(Favorite)，筛选(Filter),批量删除(deletByBatch)

3,扫描功能:采用MVVM架构，将扫描功能的业务逻辑分配给ScanViewModel,ScanViewModel负责具体的业务处理(如接受数据跳转结果页，控制缩放状态，处理闪光灯状态),采用SigleScanEvent事件总线来解决重复跳转问题默认只处理获取的最后一帧图像

4,设置功能:支持黑暗模式(Dark Mode)切换功能在应用的不同界面实现较为柔和的颜色切换

5，历史记录页面中实现单击跳转到对应的创建结果页面，长按进图编辑模式，批量删除功能和查看收藏记录项功能;

6，扫描页面支持从相册获取图片解析，闪光灯，相册图片解析失败弹窗，权限申请弹窗，和跳转扫描结果页复制文本，分享文本，滚动视图来支持长文本，短文本查看，收藏功能，自动插入历史记录等等
